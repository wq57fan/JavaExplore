# IntelliJ Idea 常用快捷键列表

```
Alt+Insert				可以生成构造器/Getter/Setter等
	
Ctrl+Alt+T				可以把代码包在一个块内，例如：try/catch;或创建单元测试用例
	
Ctrl+Shift+Alt+T		重构
	
Ctrl+Enter				导入包，自动修正
Ctrl+Alt+O 				优化导入的类和包
	
Ctrl+Alt+I				将选中的代码进行自动缩进编排，这个功能在编辑 JSP 文件时也可以工作
	
Ctrl+R					替换文本
Ctrl+F					查找文本
Ctrl+N/Ctrl+Shift+N		打开类或资源
	
Ctrl+Y					删除行
Ctrl+D					复制行
	
Ctrl+O					重写方法(hashcode,equals,toString等)
	
Ctrl+Backspace			按单词删除
Ctrl+/或Ctrl+Shift+/		注释（//或者/**/）
	
Ctrl+Shift+Up/Down		向上/下移动语句
Alt+Up/Down				在方法间快速移动定位
Ctrl+Left/Right			移动光标到前/后单词
	
Ctrl+W					自动按语法选中代码
Ctrl+Shift+W			反向按语法选中代码
	
Alt+Enter				自我修复
	
Shift+F10				Run
Ctrl+F5					ReRun
```
